# Task 3: Customer Churn Prediction

## Objective

Identify customers who are likely to leave the bank and understand the factors contributing to customer churn.

## Approach

* Cleaned and prepared the dataset.
* Removed unnecessary columns.
* Encoded categorical variables such as geography and gender.
* Performed Exploratory Data Analysis (EDA).
* Trained a classification model using Random Forest.
* Evaluated performance using accuracy and confusion matrix.
* Analyzed feature importance to identify key churn drivers.

## Results and Insights

* Customer age, account balance, and tenure influenced churn behavior.
* Feature importance analysis highlighted the most significant factors affecting customer retention.
* The model successfully identified customers at risk of leaving the bank.
* Predictive analytics can help banks improve customer retention strategies.
